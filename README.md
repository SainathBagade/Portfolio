## üöÄ Getting Started

### Prerequisites

- Node.js (v16+)
- MongoDB (Local or Atlas)

### Installation

1. **Clone the repository** (or just use the files provided).
2. **Install dependencies**:
   ```bash
   npm run install-all
   ```
   *(Note: You can run `npm install` in root, frontend, and backend separately)*

3. **Environment Variables**:
   Create a `.env` file in the `backend` folder:
   ```env
   PORT=5000
   MONGO_URI=your_mongodb_uri
   ```

4. **Run the application**:
   ```bash
   npm run dev
   ```
   This will start both the frontend (on port 5173) and backend (on port 5000) concurrently.
